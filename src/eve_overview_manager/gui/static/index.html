Overview Forge Help

Overview Editing

Import an EVE overview YAML file, edit tabs and presets, validate, then export a new YAML file. Import the exported YAML manually inside the EVE client.

Tabs And Presets

Tabs choose the active overview preset and bracket preset. Presets control visible groups and state filters. Appearance and columns are global.

Profiles

Profile tools work with local opaque EVE files. Review the dry-run plan, create a backup, close EVE, then execute only when the exact target files are correct.

Safety Rules

The app does not automate EVE, parse .dat internals, inject into the client, or overwrite exports without generating a unique filename.
